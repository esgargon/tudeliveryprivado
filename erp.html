Módulo de Servicio Técnico SAT

Mediante el módulo SAT podrás llevar la gestión integral de tu empresa de servicio técnico. Gestiona incidencias, avisos y órdenes de trabajo. Planifica contratos de mantenimiento y factúralos de forma masiva. Los técnicos acceden desde el móvil o tablet a realizar las órdenes y el cliente puede firmarlas.

15,00 € /mes

Características Principales:

  • Avisos/Incidencias
  • Presupuestos
  • Órdenes de trabajo
  • Contratos de mantenimiento
  • Artículos SAT
  • Gestión de Garantías
  • App Móvil
  • Geolocalización
  • Informes y Cuadro de Mando
  • Roles de acceso por usuario
  • Extranet de acceso a Clientes
  • Api para Desarrolladores
